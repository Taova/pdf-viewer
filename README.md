# PDF viewer
## React + TypeScript + Vite


## Prerequisites

- Node.js >= 14
- npm or yarn

## Getting Started

```bash
# Install dependencies
npm install

# Start development
npm run dev

# Build production
npm run build

# Locally preview production build
npm run preview

# Run ESLint
npm run lint
```
